## 📌 Overview
This project presents an **interactive Power BI dashboard** analyzing **US airline performance (2015)**, focusing on delays, cancellations, and operational efficiency.

The dashboard converts raw flight data into **actionable insights** for better decision-making.

---

## 🎯 Objectives
- Analyze airline on-time performance  
- Identify delay patterns  
- Compare airline efficiency  
- Evaluate route-level performance  

---

## 📊 Dashboard Sections

### 🔹 Airline Operations Overview
- On-Time Performance: **64%**
- Total Flights: **61M**
- Cancellation Rate: **2%**
- Avg Delay: **33 mins**
- Monthly trends (Flights vs On-Time %)
- Delay category breakdown
- Airline performance comparison

---

### 🔹 Route & Airport Analysis
- Top 10 busiest origin airports  
- Route performance (flights, delays, cancellations)  
- Most delayed routes  
- Average delay gauge  

---

### 🔹 Airline Performance
- 🟢 Best Airline: **Delta Air Lines Inc.**
- 🔴 Worst Airline: **Spirit Air Lines**
- ⚠️ Highest Cancellation: **American Eagle Airlines Inc.**

Includes:
- Flights vs Delay scatter plot  
- Airline performance table  
- Delay category distribution  

---

## 🧠 Key Insights
- ~64% flights are on-time  
- Minor delays contribute the highest share  
- Some routes consistently experience higher delays  
- Airline performance varies significantly
